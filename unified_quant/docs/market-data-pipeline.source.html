3. 실제 사례: HCBK 주식이 MTB 주식으로 바뀌는 과정

하나의 주식교환 합병이 공시 원문에서 백테스트 포지션 전환까지 이동한 실제 데이터입니다.

  1. 1
    마지막 가격과 기업을 찾는다EODHD의 HCBK 가격과 종목 목록을 동일한 security_id에 연결합니다.
  2. 2
    SEC 공시 원문을 내려받는다SecEdgarLifecycleSource.fetch_filing()이 공시 원문과 해시를 보관합니다.
  3. 3
    합병 조건을 구조화한다parse_sec_lifecycle_filing()이 완료일, 교환비율, 후속 티커를 추출합니다.
  4. 4
    독립 가격으로 경제성을 확인한다기존 HCBK 가치와 0.08403 × MTB 가격의 연결을 비교합니다.
  5. 5
    백테스트 포지션을 전환한다PortfolioLedger.apply_actions()가 HCBK 수량을 MTB 수량으로 바꿉니다.

  "action_type": "stock_merger",
  "effective_date": "2015-11-02",
  "security": "HCBK",
  "ratio": 0.08403,
  "new_symbol": "MTB",
  "official": true,
  "source_kind": "official_crosscheck"
}
evidence = sec_source.collect(candidate)

    evidence.content, candidate, evidence.filing
)
repository.append_frame(
    "corporate_actions",

)
